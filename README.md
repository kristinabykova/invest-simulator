# Invest Simulator

Учебный веб-проект, разработанный в рамках выпускной квалификационной работы.  
Приложение представляет собой обучающий сайт-сimulator инвестиций для начинающих пользователей и предназначено для безопасного освоения базовых принципов фондового рынка без использования реальных денежных средств.

## О проекте

Современные пользователи все чаще интересуются финансовой грамотностью и инвестициями, однако обучение только на теоретических материалах часто оказывается недостаточно наглядным. При этом реальные операции на бирже для новичков связаны с финансовыми рисками, а существующие демо-платформы нередко перегружены сложным функционалом.

**Invest Simulator** решает эту задачу, предоставляя безопасную и доступную среду для практического освоения основ инвестирования. Пользователь может анализировать поведение акций, выполнять виртуальные сделки и получать пояснения по результатам своих действий.

## Цель проекта

Разработка веб-приложения для обучения начинающих пользователей основам инвестирования с помощью симуляции инвестиционных сценариев, анализа рыночных данных и интерактивных подсказок.

## Основной функционал

- анализ выбранного временного периода на графике акции;
- расчет изменения цены на выбранном участке;
- определение направления тренда;
- оценка волатильности и уровня риска;
- формирование пояснений и обучающих подсказок;
- виртуальная покупка и продажа активов;
- регистрация пользователей;
- виртуальный инвестиционный портфель;
- встроенный словарь инвестиционных терминов.

## Особенности проекта

### Анализ периода на графике
Пользователь выбирает две точки на графике цены акции — начало и конец исследуемого периода. На основе этого интервала система выполняет анализ и формирует количественную и качественную оценку поведения цены.

Результат анализа включает:
- изменение цены;
- направление тренда;
- уровень волатильности;
- показатель риска;
- пояснение логики инвестиционного решения.

### Обучающие подсказки
Подсказки формируются двумя способами:
- с использованием искусственного интеллекта для генерации уникальных пояснений;
- с помощью заранее подготовленного набора шаблонов для гарантированного вывода корректной информации.

### Поэтапное обучение
Интерфейс ориентирован на постепенное усложнение:
- на первом этапе доступны только базовые действия;
- после освоения основ пользователь получает доступ к дополнительным инструментам управления рисками.

## Архитектура

Приложение реализовано по **клиент-серверной архитектуре**.

### Технологии backend
- **FastAPI**
- **REST API**
- **Redis** (кеширование)
- **PostgreSQL** (хранение пользовательских данных)
- **Pydantic**

### Технологии frontend
- **HTML**
- **CSS**
- **JavaScript**

### Внешние интеграции
- **API Московской биржи** для получения рыночных данных

## Практическая значимость

Проект может использоваться в образовательных целях:
- в рамках курсов по финансовой грамотности;
- при изучении основ инвестирования;
- для безопасственной отработки инвестиционных решений без риска потери реальных средств.

Приложение позволяет анализировать ошибки пользователя, формировать понимание рыночной динамики и вырабатывать более осознанное отношение к инвестиционному риску.

## Статус проекта

Проект находится в активной разработке и развивается в рамках дипломной работы.

## Структура проекта

- `backend/` — серверная часть приложения
- `frontend/` — клиентская часть приложения

## Назначение

**Invest Simulator** — это учебный инструмент, который помогает начинающим пользователям:
- понять базовые механики фондового рынка;
- научиться интерпретировать динамику цены;
- попробовать виртуальные сделки;
- получить первые практические навыки принятия инвестиционных решений.
